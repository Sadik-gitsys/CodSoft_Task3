# Spam SMS Detection

## Overview

This project is a Machine Learning model that detects whether an SMS message is **Spam** or **Legitimate (Ham)**. The model is built using **Logistic Regression** and **TF-IDF Vectorization**.

## Features

- Detects Spam and Legitimate SMS
- Uses TF-IDF for text processing
- Logistic Regression model
- Simple prediction using user input

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Joblib

## Files

- `spam.csv` - Dataset
- `train.py` - Model training
- `predict.py` - Predict new SMS
- `README.md` - Project documentation

## How to Run

1. Install the required libraries.
2. Run `train.py` to train the model.
3. Run `predict.py` to classify a new SMS message.

## Accuracy

**Accuracy:** Approximately **98%**
